# Kick 채팅 봇
구슬치기 Kick 연동과 무료 영도 기능(미구현)
경고: 아직 제대로 작동안함

# 설치

1. 이 저장소 복사
```bash
git clone https://github.com/theboocchirock/Kickbot.git
```

2. node.js 라이브러리 설치
```bash
npm install
```
3. settings.json.example 파일명을 settings.json으로 바꾸고 킥 채널 아이디 입력


4. 시작(구슬치기)
```bash
node ./chatreader/index.js
```
